Introduction To Cloud

This project was completed as part of the i2i Academy Cloud Training Program.

Objective

The objective of this assignment is to gain hands-on experience with cloud computing fundamentals by creating and managing a virtual machine on a cloud platform.

Tasks Completed

* Created an Ubuntu Virtual Machine on Google Cloud Platform (GCP)
* Verified network connectivity using a ping test
* Connected to the virtual machine via SSH
* Created a text file named hello.txt
* Verified the file contents through the Linux terminal

Environment

* Cloud Provider: Google Cloud Platform (GCP)
* Operating System: Ubuntu Linux
* Virtual Machine: Compute Engine Instance

Commands Used

Ping Test

ping <VM_PUBLIC_IP>

File Creation

echo "Hello i2i Academy!" > hello.txt
cat hello.txt

Expected Output

Hello i2i Academy!
